# NPV Calculator

A spreadsheet that models the Net Present Value of a HIL capital purchase
over a multi-year horizon.

## Inputs

- Capital cost (rack, signal conditioning, software licenses, integration)
- Recurring costs (maintenance, calibration, software renewals)
- Annual savings (defect escape reduction, regression hours saved,
  rework avoided)
- Discount rate / cost of capital
- Project horizon (years)

## Outputs

- Net Present Value (NPV)
- Internal Rate of Return (IRR)
- Simple and discounted payback period
- Sensitivity analysis on the largest savings driver
